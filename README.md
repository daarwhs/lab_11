# Лабораторная работа №11. Кострицкая Дарья

## Описание проекта: 
Одностраничный сайт о путешествии по РФ, созданный на сонове макета Figma. 
Сайт адаптирован для просмотра с различных устройств от 320 до 1280 пикселей.

## Технологии:https://github.com/daarwhs/lab_11/blob/main/README.md
JavaScript
БЭМ Nested 
Grid Layout  
Flexbox  
Медиазапросы  
Подключение шрифтов  

## Инструкция: 
В файле index.html можно найти проект, открыв его через браузер.

## Планы доработки проекта:
Расширить возможности сайта с помощью JS. 
Добавить возможность сохранять фото добавленные пользователем.
